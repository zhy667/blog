对UI的一个总结 在 JSX 的大括号内来编写 JavaScript，React 组件使用 props 来互相通信，使用 JavaScript 的 if 语句、&& 和 ? : 运算符来选择性地渲染 JSX，通过 JavaScript 的数组方法 来操作数组中的数据，从而将一个数据集渲染成多个相似的组件 部分 JavaScript 函数是 纯粹 的，这类函数通常被称为纯函数。纯函数仅执行计算操作，不做其他操作。你可以通过将组件按纯函数严格编写，以避免一些随着代码库的增长而出现的、令人困扰的 bug 以及不可预测的行为 理解组件之间的关系以及调试性能和状态管理等未来将会遇到的一些概念，以及组件的导出导入。
